## <a name="labor-cost-tracking-view"></a><span data-ttu-id="809b0-108">De weergave Kosten bijhouden voor arbeid</span><span class="sxs-lookup"><span data-stu-id="809b0-108">Labor Cost Tracking view</span></span>

<span data-ttu-id="809b0-109">Op de pagina **Projecten** op het tabblad **Bijhouden** tabblad kunt u **Bijhouden** > **Kosten** selecteren om de weergave **Kosten bijhouden** te openen en de voortgang van arbeidsuitgaven voor elke taak in het projectplan te bekijken.</span><span class="sxs-lookup"><span data-stu-id="809b0-109">On the **Projects** page, on the **Tracking** tab, you can select **Tracking** > **Cost** to open the **Cost Tracking** view and see the progress of labor spend on each task in the project plan.</span></span> <span data-ttu-id="809b0-110">Deze weergave vergelijkt ook de werkelijke arbeidskosten die aan een taak zijn besteed met de geplande arbeidskosten van de taak.</span><span class="sxs-lookup"><span data-stu-id="809b0-110">This view also compares the actual labor cost spent on a task to the task's planned labor cost.</span></span> <span data-ttu-id="809b0-111">In Project Operations worden de volgende formules gebruikt om metrische gegevens over arbeidskosten te berekenen:</span><span class="sxs-lookup"><span data-stu-id="809b0-111">Project Operations uses the following formulas to calculate labor cost metrics:</span></span>

- <span data-ttu-id="809b0-112">**Geplande kosten**: geschatte kosten van alle resourcetoewijzingen voor elke bladknooppunttaak</span><span class="sxs-lookup"><span data-stu-id="809b0-112">**Planned cost**: Estimated cost of all resource assignments on each leaf node task</span></span>
- <span data-ttu-id="809b0-113">**Werkelijke kosten**: som van alle werkelijke kosten voor de tijd die voor de taak is geregistreerd</span><span class="sxs-lookup"><span data-stu-id="809b0-113">**Actual Cost**: Sum of all cost actuals for time recorded on the task</span></span>
- <span data-ttu-id="809b0-114">**Kostenverbruikspercentage**: Werkelijke kosten ÷ Kostenschatting bij voltooiing</span><span class="sxs-lookup"><span data-stu-id="809b0-114">**Cost consumption percentage**: Actual cost ÷ Cost estimate at complete</span></span>
- <span data-ttu-id="809b0-115">**Resterende kosten**: Kostenschatting bij voltooiing - Werkelijke kosten</span><span class="sxs-lookup"><span data-stu-id="809b0-115">**Remaining Cost**: Cost estimate at complete  – Actual cost</span></span>

- <span data-ttu-id="809b0-117">**Kostenvariantie**: Geplande kosten – Kostenschatting bij voltooiing</span><span class="sxs-lookup"><span data-stu-id="809b0-117">**Cost variance**: Planned cost – Cost estimate at complete</span></span>

<span data-ttu-id="809b0-118">Elke taak toont een projectie van de kostenvariantie.</span><span class="sxs-lookup"><span data-stu-id="809b0-118">Each task shows a projection of the cost variance on the task.</span></span> <span data-ttu-id="809b0-119">Als de kostenschatting bij voltooiing hoger is dan de geplande kosten, wordt verwacht dat de taak het budget overschrijdt.</span><span class="sxs-lookup"><span data-stu-id="809b0-119">If the cost estimate at complete is more than the planned cost, the task is projected to go over budget.</span></span> <span data-ttu-id="809b0-120">Als de kostenschatting bij voltooiing lager is dan de geplande kosten, wordt verwacht dat de taak binnen het budget wordt voltooid.</span><span class="sxs-lookup"><span data-stu-id="809b0-120">If the cost estimate at complete is less than the planned cost, the task is projected to finish under budget.</span></span>

> - <span data-ttu-id="809b0-125">Op het tabblad **Schattingen** worden de geschatte kosten berekend door dezelfde valuta te gebruiken als het kostentarief in de prijslijst.</span><span class="sxs-lookup"><span data-stu-id="809b0-125">The **Estimates tab** calculates the estimated cost by using the same currency of the cost rate in the price list.</span></span> <span data-ttu-id="809b0-126">Vervolgens worden de geschatte kosten in de prijslijstvaluta geconverteerd naar de geschatte kosten in de kostenvaluta van het project.</span><span class="sxs-lookup"><span data-stu-id="809b0-126">Then, the estimated cost in the price list currency is converted to the estimated cost in the project's cost currency.</span></span> <span data-ttu-id="809b0-127">De geschatte kosten in de projectvaluta worden afgerond op twee decimalen.</span><span class="sxs-lookup"><span data-stu-id="809b0-127">The estimated cost in the project currency is shown rounded to 2 decimal places.</span></span> <span data-ttu-id="809b0-128">Tijdens deze omrekening wordt op geen enkel moment valutanauwkeurigheid toegepast.</span><span class="sxs-lookup"><span data-stu-id="809b0-128">At no point during this conversion is currency precision applied.</span></span>
