# <a name="use-the-schedule-board-to-book-project-resources"></a><span data-ttu-id="ad80f-103">Projectresources boeken door middel van het planbord</span><span class="sxs-lookup"><span data-stu-id="ad80f-103">Use the Schedule Board to book project resources</span></span>

<span data-ttu-id="ad80f-104">Naast het boeken van resources voor een project vanuit een project, kunt u resources vanuit het planbord hard boeken of zacht boeken.</span><span class="sxs-lookup"><span data-stu-id="ad80f-104">In addition to booking resources on a project from within a project, you can hard-book or soft-book resources from the Schedule Board.</span></span>

<span data-ttu-id="ad80f-105">Voordat u vanaf het planbord kunt boeken, moet u resourcevereisten maken of genereren.</span><span class="sxs-lookup"><span data-stu-id="ad80f-105">Before you can book from the Schedule Board, you must create or generate resource requirements.</span></span> <span data-ttu-id="ad80f-106">Ga als volgt te werk om resourcevereisten te maken vanuit het planbord:</span><span class="sxs-lookup"><span data-stu-id="ad80f-106">Follow these steps to create resource requirements from the Schedule Board.</span></span>

1. <span data-ttu-id="ad80f-107">Als het deelvenster **Boekingsvereisten** onderaan de pagina is samengevouwen, selecteert u het uitvouwbesturingselement om het uit te vouwen.</span><span class="sxs-lookup"><span data-stu-id="ad80f-107">If the **Booking Requirements** pane at the bottom of the page is collapsed, select the expander control to expand it.</span></span>
2. <span data-ttu-id="ad80f-108">Selecteer in het deelvenster **Boekingsvereisten** op het tabblad **Project** de vereiste die u wilt boeken.</span><span class="sxs-lookup"><span data-stu-id="ad80f-108">In the **Booking Requirements** pane, on the **Project** tab, select the requirement to book.</span></span>

    ![Vereiste geselecteerd op het tabblad Project](media/Resource-Management-image73.png)

3. <span data-ttu-id="ad80f-110">Selecteer **Beschikbaarheid zoeken** om de te reserveren resources te filteren en de beschikbare resources weer te geven.</span><span class="sxs-lookup"><span data-stu-id="ad80f-110">Select **Find Availability** to filter the bookable resources and view the available resources.</span></span> 
4. <span data-ttu-id="ad80f-111">Selecteer een of meer resources op het planbord.</span><span class="sxs-lookup"><span data-stu-id="ad80f-111">Select one or more resources from the Schedule Board.</span></span> 
5. <span data-ttu-id="ad80f-112">Voer in het deelvenster **Resourceboeking maken** aan de rechterkant van de pagina de boekingsgegevens in en selecteer **Boeken en afsluiten.**</span><span class="sxs-lookup"><span data-stu-id="ad80f-112">In the **Create Resource Booking** pane on the right side of the page, enter the booking information, and then select **Book and exit**.</span></span>

    ![Deelvenster Resourceboeking maken voor de geselecteerde boekbare resource](media/Resource-Management-image74.png)

6. <span data-ttu-id="ad80f-114">Terwijl de vereiste is geselecteerd in het deelvenster **Resourceboeking maken**, selecteert u een of meer cellen van een resource om de boeking te maken.</span><span class="sxs-lookup"><span data-stu-id="ad80f-114">While the requirement is selected in the **Create Resource Booking** pane, select one or more cells of a resource to create the booking.</span></span>

    ![Meerdere cellen geselecteerd voor een resource](media/Resource-Management-image75.png)

7. <span data-ttu-id="ad80f-116">Selecteer **Boeken**.</span><span class="sxs-lookup"><span data-stu-id="ad80f-116">Select **Book**.</span></span>

<span data-ttu-id="ad80f-117">De vereiste wordt vervuld met de geselecteerde resource.</span><span class="sxs-lookup"><span data-stu-id="ad80f-117">The requirement is fulfilled by using the selected resource.</span></span> <span data-ttu-id="ad80f-118">In het deelvenster **Reserveringsvereisten** ziet u dat de vereiste is bijgewerkt en de resource wordt weergegeven als geboekt voor het project.</span><span class="sxs-lookup"><span data-stu-id="ad80f-118">In the **Booking Requirements** pane, notice that the requirement has been updated, and the resource is shown as booked on the project.</span></span>

![Resource is geboekt voor het project](media/Resource-Management-image76.png)
