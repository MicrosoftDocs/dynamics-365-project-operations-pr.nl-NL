<span data-ttu-id="88646-110">Een subset van de opgenomen onderdelen kan worden gemarkeerd als toerekenbaar met het veld **Factureringstype**.</span><span class="sxs-lookup"><span data-stu-id="88646-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="88646-111">Het veld **Factureringstype** is een optieset die de volgende waarden toestaat in de context van een contractregel:</span><span class="sxs-lookup"><span data-stu-id="88646-111">The **Billing Type** field is an option-set that allows the following values in the context of a contract line:</span></span>
